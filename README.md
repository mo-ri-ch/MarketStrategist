# Competitor Intelligence AI Assistant

A multi-tenant AI-powered SaaS platform that monitors competitors, discovers market opportunities, tracks social/media/news activity, and generates CEO-level strategic recommendations.

## Repository Structure

- `frontend/` - Next.js web application built with Tailwind CSS and TypeScript.
- `backend/` - FastAPI backend service handling core logic, database communication, and asynchronous workers.
- `agents/` - Multi-agent workflows built using LangGraph.
- `docker/` - Docker configuration files for development and production environments.

## Getting Started

Refer to the individual service directories for specific setup and configuration instructions.
